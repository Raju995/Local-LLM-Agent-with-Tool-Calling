# Local-LLM-Agent-with-Tool-Calling
<h1 align="center" id="title">Local LLM Agent with Tool Calling</h1>

<p id="description">LLM will call local agent based on the user query if LLM has no information about the question then it will do web search</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Install Python</p>

<p>2. Create Virtual Environment</p>

```
python3 -m venv venv
```

<p>3. Activate virtual Environment</p>

```
source venv/bin/activate
```

<p>4. Install Requirements</p>

```
pip install -r requirements.txt
```

<p>5. Run Migrations</p>

```
python manage.py makemigrations
```

<p>6. Run Migrate</p>

```
python manage.py migrate
```

<p>7. Run The server</p>

```
python manage.py runserver
```
